# **PSU_zappy_2017**

> Just another useless school project

## Usage :

-   `make zappy_server && make clean` to build

-   See required flags and arguments with `./zappy_server -h`

-   `./zappy_server -[flag] [options]` to run

-   Connect your  **ia client**

-   Enjoy
